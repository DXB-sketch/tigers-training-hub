# Bribie Island Tigers FC — Training Hub

Training plan management app for Bribie Island Tigers FC.

## Development
See CLAUDE.md for architecture and conventions.
See /mockups for visual references for every screen.

## Setup
1. Clone the repo
2. Run `npm install`
3. Copy `.env.local.example` to `.env.local` and add Supabase credentials
4. Run `npm run dev`
